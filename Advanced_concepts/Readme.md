# Concept 1: OOP — Classes, Instance & Class Variables, Class & Static Methods

## What I Learned / Implemented

- Created a **class** in Python with:
  - **Instance variables:** unique to each object (`first`, `last`, `pay`, `date`, `email`)
  - **Class variables:** shared across all objects (`no_of_emp`, `raise_amount`)
- Implemented **methods**:
  - Regular method (`apply_raise`) to operate on instance variables
  - **Class method** (`from_string`) to create objects from strings
  - **Class method** (`apply_raise_classmethod`) to modify class variables
  - **Static method** (`is_workday`) that doesn’t access class or instance variables
- Learned the difference between **instance and class variables**, and how **objects behave individually** vs shared class-level attributes

## Key Python Concepts Practiced

- Object-Oriented Programming (OOP)
- Instance vs Class Variables
- Regular Methods vs `@classmethod` vs `@staticmethod`
- Alternative constructors with class methods
- Using `datetime` module for utility functions
